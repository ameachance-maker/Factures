# 🧾 Générateur de Factures & Devis

Une application web simple et rapide pour créer des documents professionnels comme des factures ou des devis, avec export en PDF.

---

## 🎯 Objectif

Ce projet vise à simplifier la création de documents commerciaux pour les indépendants, artisans et petites entreprises. Il permet de générer des factures claires, personnalisées et prêtes à être envoyées.

---

## ⚙️ Technologies utilisées

- **Python** avec **FPDF** pour la génération de fichiers PDF
- **Flask** pour le backend web
- **HTML, JavaScript & Bootstrap 5** pour une interface moderne et responsive

---

## ✨ Fonctionnalités

- Ajout dynamique de produits/services
- Calcul automatique des totaux (HT, TVA, TTC)
- Export PDF prêt à l’envoi
- Interface utilisateur élégante et intuitive
- Ajout de logo (optionnel)
